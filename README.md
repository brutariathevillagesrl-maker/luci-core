# LUCI Core (Layer 1 Blockchain)

*LUCI* este o rețea descentralizată de tip Layer 1, construită pentru eficiență și securitate, utilizând un mecanism hibrid de minare.

## 🛠 Specificații Tehnice
*   *Algoritm Hibrid:* KawPow (GPU) & GhostRider (CPU)
*   *Consens:* Proof of Work (PoW)
*   *Timp pe bloc:* 60 secunde (estimat)
*   *Unitate de măsură:* Bees
*   *Conversie:* 1 LUCI = 100,000,000 Bees

## 🐝 Unitatea "Bees"
Sistemul monetar al LUCI folosește unitatea *Bees* pentru micro-tranzacții. 
- 1 Bee = 0.00000001 LUCI
- 100,000,000 Bees = 1.00000000 LUCI

## ⛏ Mineze (Mining)
Rețeaua LUCI permite utilizatorilor obișnuiți să mineze folosind:
1. *GPU (Plăci Video):* Algoritmul KawPow.
2. *CPU (Procesoare):* Algoritmul GhostRider.
